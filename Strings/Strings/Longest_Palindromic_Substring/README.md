# Longest Palindromic Substring

## Problem Statement

Given a string, find the longest palindromic substring present in it.

## Approach

* Generate all possible substrings.
* Check each substring using two pointers.
* Track the longest palindrome found.

## Time Complexity

O(n³)

## Space Complexity

O(1)

## Concepts Used

* Strings
* Nested Loops
* Two Pointers
* Brute Force

## Language

C
